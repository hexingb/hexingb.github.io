Ubuntu 切换 Desktop Environment 到 Window Manager

很久很久以前，我是直接使用 Xorg + 窗口管理器的。电脑启动后，直接进入黑乎乎的 Console，login 输入用户名和密码，然后执行 startx，或者先进一点，加一个 lightdm。岁数大了就不想折腾，公司电脑是 Mac，所以家里性能拉垮的笔记本就一直是 Ubuntu Gnome。现在失业了，就简单弄一下，毕竟自己还是习惯平铺窗口和键盘操作。

直接安装 spectrwm： 

sudo apt install spectrwm
cp /etc/spectrwm.conf ~/.spectrwm.conf

简单配置一下就能用了。直接在 GDM 登录的时候，选择 spectrwm 即可。

刚开始没找到输入法。因为自己的输入法快捷键和 spectrwm 的 layout 切换键冲突了。通过 ibus-setup 修改一下即可。一个让我比较开心的是，输入法切换时，在桌面中央有了明显的图标提示，比 Gnome 显示在细小的 top bar 好。

另外的问题是，触摸板无法点触做单击了。上网搜了一下，找到这个。照做，把下面两行写入 /etc/X11/xinit/xinputrc 或者 ~/.xinputrc ： 

xinput set-prop 11 "libinput Tapping Enabled"  1
xinput set-prop 11 "libinput Natural Scrolling Enabled" 1

分别表示开启 libinput Tapping Enabled 和 libinput Natural Scrolling Enabled 。

年轻的时候不懂事，喜欢折腾各种皮毛的技术。还记得上学时折腾各种发行版，配置 fvwm 之类各种窗口管理器。不如踏实学一些实用的东西。
