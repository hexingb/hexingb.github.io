关于我

一个普通的程序员，主要写 C/C++，偶尔写其它。年逾不惑，以开发、架构技术糊口并自娱，因此是个无趣的人儿。Emacs、命令行、Linux 以及类似风格的东西都是躲不掉的标签。读书、编码、下棋、俯卧撑、睡懒觉。哈哈。

关于这些页面

看书、编码的过程中遇到的问题，大多数能在网上找到浅显直白的答案。我不觉得那些源自网络的东西值得再发布到网络里去。

不过，纸上得来终觉浅。不常用、不深入的东西总是会遗忘；零散的知识不成体系，也不深入。我把自己学习笔记中的一些内容整理成了这些页面，督促自己深入底层原理，构建完整知识体系。

当然，我会很高兴知道有人觉得这些东西有帮助。

这些页面是我用 Emacs 写的 org mode 文档笔记，通过 org mode export 为 html 发布，加上一些手写的 CSS 样式得到。
